# Utility Tools

ブラウザだけで動く、エンジニア向けの静的ユーティリティ集です。

GitHub Pages などの静的ホスティングに置くだけで使えます。ビルドやサーバーは不要です。

## 特徴

- すべて静的HTML/CSS/JavaScriptで動作
- npm、外部CDN、外部APIに依存しない
- Service Workerでオフライン利用に対応
- 入力した文字列や選択したファイルはブラウザ内だけで処理
- 外部通信を禁止するCSPと、通信APIのブロックログを画面に表示

## 機能

- JSON整形: 整形、圧縮、キーソート、JSON文字列の展開
- JWT解析: ヘッダーとペイロードのデコード
- Base64変換: エンコード、デコード、URL-safe変換、ファイル変換
- 文字列変換: UUID、URL encode/decode、HTML escape/unescape、ケース変換、SHA-256
- Markdownエディター: プレビュー、表、コードブロック、Mermaid風フローチャート簡易表示
- ランダム生成: パスワード、トークン、数値、色、Lorem、CSV
- 画像・動画・音声変換: 画像のPNG/JPEG/WebP/GIF/ICO変換、画像圧縮、短い動画からGIF生成
- 画像をICOに変換: 既存画像からfavicon.icoを作成
- QRコード生成: 短い文字列やURLからQRコードを生成
- 時刻変換: Epoch秒/ミリ秒、タイムゾーン変換
- データ保護のしくみ: 外部通信ブロック状態とログの確認

## 使い方

`index.html` をブラウザで開くだけで使えます。

GitHub Pagesで公開する場合は、このリポジトリの `main` ブランチをPagesの公開元に設定してください。

公開ページ: https://lirlia.github.io/utility-tools/

`sw.js` はオフライン利用とキャッシュ更新のために使います。通常の静的サイトとして表示するだけなら必須ではありませんが、このツールでは「一度開いた後にオフラインでも使える」状態にするため残しています。

## データ保護について

このツールは、貼り付けたデータや選択したファイルをサーバーへアップロードしません。

画面内の処理はブラウザ上で完結します。さらに、ページには `connect-src 'none'` のCSPを設定し、`fetch`、`XMLHttpRequest`、`WebSocket`、`sendBeacon` もページ内でブロックしています。通信しようとした場合は「データ保護のしくみ」画面に記録されます。

## 制限

- JWTの署名検証は行いません。
- MarkdownのMermaid対応は簡易描画です。Mermaid.js完全互換ではありません。
- QRコード生成はオフライン実装のため、短い文字列やURL向けです。
- 動画からGIFへの変換はブラウザ内処理のため、長い動画や高解像度動画では重くなります。
- GIF以外の動画・音声の本格的な形式変換には対応していません。
